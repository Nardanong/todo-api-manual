Clone the project
https://github.com/Nardanong/todo-api-manual.git

Go to the project directory
cd todo-api-manual

Initialize a new Node.js project:
 npm init -y

Install the packages
npm install / npm i

Install necessary dependencies:
npm install express body-parser ejs

Start the Server
npm start
